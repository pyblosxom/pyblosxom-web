<p>
This plugin allows you to use a different format for your weblog entries. 
This format is basically a simple python file, similar to the config.py file. 
For example:
</p>
<pre>
title="Example Entry"
date="2004-06-30-00-00"
body="""
&lt;p&gt;This is the body of the example entry.&lt;/p&gt;
"""
</pre>
