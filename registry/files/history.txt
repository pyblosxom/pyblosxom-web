<p>
  <b>history</b> is a PyBlosxom plugin that displays change history for 
  entries.  The change history must already exist, either in a version 
  control system, database, or another form.  This plugin just uses that 
  backend to display change history, previous versions of entries, and 
  textual diffs for changes.
</p>
