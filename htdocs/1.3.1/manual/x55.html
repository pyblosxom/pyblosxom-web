1.3. Where to go for help

The PyBlosxom project web-site is at http://pyblosxom.sourceforge.net/ . There you can find up-to-date documentation, manuals, project status and mailing list information.

The pyblosxom-users mailing list is the place to go if you're having difficulties. Discussions from the mailing list affect the content of this manual.
